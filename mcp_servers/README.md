# Model Context Protocol (MCP) Servers

This directory contains various Model Context Protocol (MCP) server implementations, enabling AI agents to interact with external services and data sources.

## Components:

### GitHub MCP Server
- **Source**: `tywade1980/github-mcp-server`
- **Description**: Connects AI tools directly to GitHub's platform, allowing agents to read repositories, manage issues and PRs, analyze code, and automate workflows through natural language interactions.
- **Key Files/Folders**: The entire `github-mcp-server` repository is included.

### Playwright MCP
- **Source**: `tywade1980/playwright-mcp`
- **Description**: Provides browser automation capabilities using Playwright, enabling LLMs to interact with web pages through structured accessibility snapshots.
- **Key Files/Folders**: The entire `playwright-mcp` repository is included.

### Hugging Face MCP Server
- **Source**: `tywade1980/hf-mcp-server`
- **Description**: Connects LLMs to the Hugging Face Hub and thousands of Gradio AI Applications.
- **Key Files/Folders**: The entire `hf-mcp-server` repository is included.

### Firecrawl MCP Server
- **Source**: `tywade1980/firecrawl-mcp-server`
- **Description**: Integrates with Firecrawl for web scraping, crawling, and content extraction capabilities.
- **Key Files/Folders**: The entire `firecrawl-mcp-server` repository is included.

### Zapier MCP
- **Source**: `tywade1980/zapier-mcp`
- **Description**: Enables AI assistants to access over 8,000 apps and 30,000 actions via Zapier, facilitating real-world actions through natural language commands.
- **Key Files/Folders**: The entire `zapier-mcp` repository is included.

### monday.com MCP
- **Source**: `tywade1980/mcp-monday`
- **Description**: monday.com's open framework for connecting AI agents into their work OS, providing secure access to structured data, tools for action, and context for decision-making.
- **Key Files/Folders**: The entire `mcp-monday` repository is included.
