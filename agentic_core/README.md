# Agentic AI & Reasoning Foundations

This directory contains the consolidated code for core agentic AI and reasoning capabilities, drawing from various repositories.

## Components:

### Action (Multi-Agent Systems in Python)
- **Source**: `tywade1980/action`
- **Description**: Provides foundational elements for building autonomous, multi-agent systems in Python. Includes examples for deep research, engineering teams, and trading floor simulations.
- **Key Files/Folders**: `action_deep_research/`, `action_engineering_team/`, `action_trading_floor/`, `action_pyproject.toml`, `action_uv.lock`

### Hierarchical Reasoning Model (HRM)
- **Source**: `tywade1980/HRM`
- **Description**: A novel recurrent architecture for complex goal-oriented reasoning, inspired by hierarchical processing in the human brain. Achieves significant computational depth with minimal data.
- **Key Files/Folders**: `hrm_models/`, `hrm_utils/`, `hrm_evaluate.py`, `hrm_pretrain.py`, `hrm_requirements.txt`

### Caroline Alpha (Advanced AI Assistant)
- **Source**: `tywade1980/caroline-alpha`
- **Description**: An advanced AI assistant framework, providing core functionalities for intelligent assistance.
- **Key Files/Folders**: `caroline_soul_core_pack/`, `caroline_app.py`, `caroline_cli.py`, `caroline_config.py`, `caroline_requirements.txt`

### Kilo Code (Open-source VS Code AI Agent)
- **Source**: `tywade1980/kilocode`
- **Description**: An open-source VS Code AI agent for code generation, task automation, and browser automation. Integrates with various AI models and MCP servers.
- **Key Files/Folders**: `kilocode_src/`, `kilocode_package.json`

### Mem0 (Long-Term Memory for AI Agents)
- **Source**: `tywade1980/mem0` and `tywade1980/mcp-mem0`
- **Description**: Enhances AI assistants and agents with an intelligent memory layer, enabling personalized AI interactions. Supports multi-level memory (User, Session, Agent state) and various LLMs.
- **Key Files/Folders**: (Code will be integrated directly, no specific subfolder for this consolidation step)

### Ollama (Local LLM Execution)
- **Source**: `tywade1980/ollama`
- **Description**: Facilitates getting up and running with large language models locally.
- **Key Files/Folders**: (Code will be integrated directly, no specific subfolder for this consolidation step)
